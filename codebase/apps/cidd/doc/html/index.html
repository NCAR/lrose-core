# Last Update:  April 2001 - Frank Hage NCAR/RAP (fhage@ucar.edu)

Configurable, Interactive Data Display (CIDD)
Documents, Information and Examples

One of the unfortunate things about CIDD is its lack of good, comprehensive documentation. CIDD has been under development off and on since March of 1991 and RAP's projects have always chosen to fund feature additions rather than documentation. Please look at the following documents, or go through some of the sample configuration files and then call me at 303-494-8413 if you need additional information or help.

- Frank Hage (fhage@ucar.edu)

Currently Available Documentation

  • CIDD Introduction - A text overview of what CIDD's capabilities are.
  • A Manual written for the Taiwan project This manual is specific to the configuration of a display system as configured for Civil Aviation Operations Center Users in Taiwan. It has the most up-to-date information about CIDD's menus and control panels. It also shows some examples of how CIDD works with ancillary applications which display tabular data nearest the last click point. Here's a direct link to the CIDD section of the manual.
  • CIDD MAP File Format - A Brief Guide to CIDD's MAP file format - with example links.
  • CIDD README - Some information which discusses CIDD's components and configuration file.
